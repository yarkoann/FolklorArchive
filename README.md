# ФонограммАрхив - Система метаданных

Система метаданных для цифрового архива фонограмм с поддержкой онтологии музыкального фольклора народов Арктики.

## Возможности

- 📀 Хранение метаданных аудиозаписей фольклора
- 🌍 Поддержка контролируемого словаря терминов арктических культур
- 🔒 Этические флаги доступа (публичный/ограниченный/требует согласия общины)
- 🔍 Фасетный поиск с фильтрацией по этносу, жанру, статусу
- 📤 Экспорт в формат METS/ALTO
- 🌐 Веб-интерфейс для работы с архивом

## Требования

- Python 3.8+
- Установленные зависимости из requirements.txt

## Установка

1. Клонировать репозиторий:
```bash
git clone https://github.com/yarkoann/FolklorArchive
cd folklor_archive
```
2. Создать виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
venv\Scripts\activate     # для Windows
```
3. Установить зависимости:
```bash
pip install -r requirements.txt
```
4. Запуск:
```bash
python app.py
```
